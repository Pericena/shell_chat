 ### instalacion termux
 ```
 pkg install php
 pkg install openssh
 pkg install git
 git clone https://github.com/Pericena/shell_chat.git
 cd shell_chat
 chmod 711 shell.sh
 ./shell.sh
 bash shell.sh
 ```
 
  ### instalacion linux
